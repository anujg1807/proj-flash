Drop your resume(s) here as plain .txt files, one per variant.

File naming convention:
  general-pm.txt
  technical-pm.txt
  growth-pm.txt
  etc.

Each file is scored against every new job and the best-matching
resume variant is shown in the Telegram notification:

  🎯 Fit: Strong — AI tooling domain matches, 5y exp meets req (technical-pm)

Requirements:
- Files must end in .txt
- This README.txt file is ignored
- ANTHROPIC_API_KEY must be set in GitHub Actions secrets for scoring to work
- If no resume files are present or API key is missing, fit scoring is silently skipped
